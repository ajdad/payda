# Payda.ma Prototype

This repository contains a simple prototype for Payda.ma, a Moroccan platform for sharing verified reviews about online sellers.

## Features

- Homepage with search bar and recent reviews
- Store pages showing average rating and reviews
- Review submission form
- Seller dashboard and admin panel placeholders
- Express-based API with in-memory data

## Running the prototype

1. Install Node.js and npm.
2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the server:

   ```bash
   npm start
   ```

4. Open `http://localhost:3000` in your browser.

This prototype uses mock data and does not include authentication or database integration.
